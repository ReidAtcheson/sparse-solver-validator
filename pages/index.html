Sparse Solver Validator

source code

Description

Sparse Solver Validator is a Rust toolkit for generating public sparse linear systems and validating a solver-owned solution. Problems are reproducible from a versioned generator. Validation can run locally, or a server can issue a signed, expiring challenge and return a signed certificate for a submitted proof.

This is development software and has not received an external security or numerical audit. The direct backend reveals the complete solution; the succinct backends do not claim zero knowledge. See the repository documentation for protocol details and limits.

Usage

The hosted service exposes:

POST ${SERVICE_URL}/v1/challenges
POST ${SERVICE_URL}/v1/validate
GET  ${SERVICE_URL}/health

Start with a problem template, ask the service to bind fresh signed randomness to it, finalize the public matrix, solve it, create a proof, and submit that proof for a signed certificate. The explorer below generates the exact template and commands for both this hosted workflow and a completely local workflow.

A minimal request/submit sequence looks like this (add an identity-token header for a private Cloud Run service):

export SERVICE_URL="https://YOUR-SERVICE-URL"

curl --fail --silent --show-error \
  -H 'content-type: application/json' \
  --data-binary @/tmp/template.json \
  "${SERVICE_URL}/v1/challenges" \
  -o /tmp/challenge.json

# Finalize the challenge, solve the generated system, and create validation.proof.

curl --fail --silent --show-error \
  -H 'content-type: application/octet-stream' \
  --data-binary @/tmp/validation.proof \
  "${SERVICE_URL}/v1/validate" \
  -o /tmp/certificate.json
